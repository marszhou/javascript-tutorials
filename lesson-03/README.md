## 函数基本定义

1. 函数是Javascript程序的基本组成部分。
2. 在Javascript中函数是用来执行一个过程或者运算一个数据结果的一堆语句的集合。
3. 如要使用函数，则需要在某个作用域中先定义它，然后再调用它。

**函数也是一个对象**
