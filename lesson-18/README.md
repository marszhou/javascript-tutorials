## 算法题

写一个函数，该函数只接受一个参数n，n为数字，计算n转换成2进制后数字中『1』的个数。（暂时不考虑负数）
如函数形式为：

```
function f(n) {
  // ...逻辑
}

f(4) // 4 -> 二进制100，应返回1
f(5) // 5 -> 二进制101，应返回2
f(11) // 11 -> 二进制为1011，应返回3
```
